# password-strength-checker-gui
A simple Python GUI application that checks password strength 
based on security best practices.
## Features
- Checks minimum length (8+ characters)
- Validates uppercase and lowercase letters
- Detects numbers
- Detects special characters
- GUI built using Tkinter
- Color-coded strength result
## Tools Used
- Python
- Tkinter
- Regular Expressions (re module)
- ## How to Run

Clone the repository:

git clone https://github.com/nain_0003/password-strength-checker-gui.git

Run the file:

python password_checker_gui.py

## Purpose

Built to demonstrate basic cybersecurity validation logic and GUI development skills.
